# EmmaGetsHired 🌿
 
A gamified job search tracker that makes the job hunt a little more fun. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.
 
🔗 **Live demo:** [emmawichmann.github.io/emmagetshired](https://emmawichmann.github.io/emmagetshired)
 
---
 
## What It Does
 
Every time you apply to a job, you earn a coin. Use those coins to buy seeds, grow a garden, and collect surprise animal visitors along the way.
 
- **📋 Application Tracker** — logs every application with timestamps, grouped by day
- **💰 Coin System** — earn 1 coin per application, bonus coins at milestones (10, 25, 50, 100+ apps)
- **🌾 Seed Shop** — 40+ seeds across 6 categories (flowers, fruits, veggies, trees, herbs, rare)
- **🌱 Interactive Garden** — drag and drop seeds to plant them, watch them grow in real time
- **🐾 Animal Visitors** — rare animals visit your garden at milestone achievements (Common → Legendary)
- **↩ Undo/Redo** — accidentally clicked? undo your last application
- **🎉 Confetti** — because every application deserves a celebration
---
 
## Tech Stack
 
- HTML5
- CSS3 (custom properties, gradients, animations)
- Vanilla JavaScript (no frameworks)
- localStorage for persistent data
- Web Audio API for sound effects
- Canvas API for confetti
